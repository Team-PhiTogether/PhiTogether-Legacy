## 🚀 部署自己的 PhiTogether 实例

**仅使用 Linux 进行基础部署示范。**

1. 下载此存储库到生产环境

```bash
git clone https://github.com/Team-PhiTogether/PhiTogether && cd PhiTogether
```

2. 安装环境、编译后端

- [在你的操作系统上安装 Go 环境](https://golang.google.cn/doc/install)。

安装完成后执行：

```bash
go build main.go
```

3. 根据文件中注释配置 `config.json` (配置完后注释需删除)

4. 运行

```bash
./main
```
